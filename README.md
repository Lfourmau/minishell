# minishell

## specifications
Creating a small bash-like shell.
Main features are:
 * pipes
 * redirections,
 * variable expansion
 * implementing the built-ins **echo**, **cd**, **pwd**, **env**, **unset**, **export**, **exit**.

Project made in team of 2, big up @Ncatrien
