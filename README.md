# minishell

## Goal
Creating a small bash-like shell.
Main features are:
 * pipes
 * redirections,
 * variable expansion
 * implementing the built-ins **echo**, **cd**, **pwd**, **env**, **unset**, **export**, **exit**.

## Technos
C, Makefile

## Skills
Forks, Pipes, Process management, File system, Environment variables, Signals
